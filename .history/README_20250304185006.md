# Auction_web

## Prerequisites
- Node.js
- npm or yarn
- Any other dependencies required by your project

## Setup
1. Clone the repository:
    ```bash
    git clone <repository-url>
    cd Auction_web
    ```

2. Install dependencies for both frontend and backend:
    ```bash
    cd frontend
    npm install
    cd ../backend
    npm install
    ```

## Build
1. Build the frontend:
    ```bash
    cd frontend
    npm run build
    ```
